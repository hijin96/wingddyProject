# Wingddy
- 파이널 프로젝트
- 초등학생 영어교육 학습 사이트
- 학습 보상을 위한 선생님 스토어
## 개발환경
### Operating System
 Windows 11 64bit
### Language
Java 8, HTML5, CSS, XML, JavaScript
### WAS
Apache Tomcat 8.5
### Database
ORACLE 11g XE
### API 
행정안전부 API, ckeditor
### Library
Ajax, jQuery, ojdbc6,
json-simple-1.1.1, JSTL 1.2.5
### Framework
MyBatis 3.5.13,
Spring Framework 5.3.22
### IDE 
STS 3.9.13 release

## 구현기능
- 계정관리
01 구현기능
1) 카카오 API를 통한 소셜 로그인
2) 로그인, 회원가입
3) 회원정보 수정
1) 클래스 입장 및 신청 수락
2) 학생 학습콘텐츠(단어장 등록, 수정, 삭제)
3) 파파고API를 통한 단어검색
4) 학생일정캘린더 등록,조회,수정
5) 마니또 관리 매칭
6) 쪽지
- 클래스 및 학생 관리
1) 알파벳 랜덤 뽑기 (칭찬스티커개념)
2) 보유 알파벳 단어완성
3) 알파벳 마켓

- 선생님 스토어
1) CK에디터를 이용한 상품스토어
게시글 등록,수정,삭제
2) 장바구니 등록 ,수정 ,삭제
3) 구매자 리뷰 작성
4) 위시리스트 등록/삭제
5)상품검색
